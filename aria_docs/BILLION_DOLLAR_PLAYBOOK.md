## Data Assets to Build (Start Now, Compound Over Time)

### Asset 1: Call Pattern Database
- Every call transcript (anonymized) → what do callers ask?
- Per-vertical: top 20 questions for barbershops, dentists, vets, etc.
- Use to: Pre-build better agent templates, reduce setup time
- **Aria's role:** Aggregate call patterns monthly from Supabase, categorize by vertical

### Asset 2: Conversion Intelligence
- Which cold email subjects get opens?
- Which pitch angles close which verticals?
- Which demo scripts convert best?
- Use to: Continuously optimize sales machine
- **Aria's role:** Track all outreach metrics, identify winning patterns, recommend changes

### Asset 3: Churn Prediction Model
- Which client behaviors predict churn?
- Call volume decline + email non-opens + support tickets = churn risk
- Use to: Intervene before clients leave
- **Aria's role:** Calculate health scores, detect patterns, alert early

### Asset 4: Agent Performance Benchmarks
- How does each agent perform vs. peers in the same vertical?
- Average calls/day, average duration, resolution rate
- Use to: Show prospects "agents in your vertical handle X calls/day"
- **Aria's role:** Generate quarterly benchmark reports

### Network Effect Triggers
- **Direct:** Each new client's data improves templates for all future clients in that vertical
- **Indirect:** Each CertusOrdo customer validates the governance model for all prospects
- **Cross-side:** Developers building on CertusOrdo create agents that run on InSync's voice platform
